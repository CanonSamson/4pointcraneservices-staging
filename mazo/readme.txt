== Description ==
Our mazo theme is designed to show off the power of Mazo personal blog. It features custom styles for all the default blocks, and is built so that what you see in the editor looks like what you'll see on your website. Mazo is designed to be adaptable to a wide range of websites, whether you’re running a photo blog, launching a artist, or supporting a non-profit. Featuring ample whitespace and modern sans-serif headlines paired with classic serif body text, it's built to be beautiful on all screen sizes.
